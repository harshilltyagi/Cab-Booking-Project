# 🚖 Cab Booking System

A full-stack web application that allows users to book cab rides, manage trips, and track ride history.

---

## 📌 Features

- User registration and login
- Book cab rides
- View ride history
- Driver management
- Admin dashboard

---

## 🛠 Tech Stack

Frontend
- REACT
- Tailwind
  

Backend
- Node.js
- Express.js

Database
- Supabase

Tools
- GitHub

## ⚙ Installation

Clone the repository


git clone https://github.com/harshilltyagi/Cab-Booking-Project.git


Go to project folder


cd Cab-Booking-Project


Install dependencies


npm install


Run project


npm start
